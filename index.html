About me

Growing up, I was a curious coder who was fascinated by the abstract thinking needed for computer science. I started with Java and C++ more than ten years ago and have since added JavaScript, React, Redux, Ruby on Rails and SQL to my technical stack.

I enjoy the process of constructing applications, implementing new technologies, and collaborating with others. I continuously strive to add to my stack by learning new languages and frameworks, but I stay mindful of mobile-first design, user experience, and backwards compatibility.

Jamify

A single-page Spotify clone where users can stream music and create playlists.

  • Integrated HTML5 Audio Element with React Component State to design a custom audio player to provide a seamless listening experience for the user.
  • Incorporated AWS S3 for media storage and organization, allowing for scalability of the application and reducing the server load.
  • Established RESTful routes to perform CRUD actions on playlists, allowing users to save favorite songs for a personalized experience.

NYS Budget

A responsive single page JavaScript app that visually represents NYS Government’s Budgets.

  • Implemented D3’s graphical components and D3’s CSV manipulation to provide a visualized representation of New York State’s budget from year to year.
  • Designed with a responsive, mobile-first approach to allow users to view the web app with any device.
  • Utilized vanilla JavaScripts’s DOM manipulation to integrate a popup feature that provides more information on a fund component to help users comprehend the information better.
